#  Whatsapp Chat Analyzer using NLP

- The WhatsApp Chat Analyzer is a web application built using NLP techniques that provides a comprehensive analysis report of WhatsApp chats, both for individual users and groups.

- The report includes valuable insights such as total messages sent, total number of words, links shared, and more. Additionally, the tool highlights the most
active users, the most commonly used words and provides a sentiment analysis.
### Requirements:
- streamlit
- matplotlib
- seaborn
- urlextract
- wordcloud
- pandas
